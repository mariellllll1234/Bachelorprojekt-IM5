# Bachelorprojekt-IM5

Mein Bachelorprojekt in IM5

In meiner Bachelorarbeit werde ich mich dem Thema "Reality Dating Formate" annehmen und habe passend dazu, für dem Leistungsnachweis IM5, eine Webseite erstellt, die einige Informationen rund um das Thema zusammenfasst. Einerseits gibt es eine Seite die sich mit den Auswirkungen kritisch auseinandersetzt und auf der anderen Seite gibt es einen Überblick über deutschsprachige RTV-Formate aus dem Spekturm Dating. Zusätzlich habe ich noch die Seite "Konsumspirlae" ergänzt, da ich die Animtion toll fand - auch wenn es der Seite keinen Mehrwert bietet... aber wir sollten ja Sachen ausprobieren.

Vorgehensweise
Ich habe die Grundbausteine der Webseite mittels ChatGPT erstellt. Zuerst wurde es ein OnePager der einer - ich zitiere Wolfgang Bock - "Text-Wüste" mutierte. So beschloss ich also den Content auf zwei Seiten aufzuteilen und den Text zu kürzen. Danach begann ich interaktive Elemente mittels JS und CSS einzubauen wie beispielsweise der Typewriter im header-Bereich.

Schritt für Schritt ging ich Layout und Usability durch und kam langsam zu meinem Endprodukt. 

Challenges & Learnings
ICh bin positiv beeindruckt was man alles mithilfe KI erreichen kann. Mir ist allerdings auch bewusst geworden, dass ohne das Wissen aus den letzten Semestern IM die Arbeit mit ChatGPF auch komplizierter machen könnte. Es ist so, dass der Code nicht immer funtioniert - gerade bei JS ist das Problem häufig aufgetreten. Da ist es natürlich sinnvoll wenn man in etwa den Code versteht und dann aus eigenem Wissen den COde anpassen kann. So konnte ich mir beispielsweise einige Male selber helfen. Das Problem mit ChatGPF lag für mich unteranderem auch darin, dass ich immer sämtlichen bestehenden Code in den CHat kopieren musste, damit die KI auf diesem aufbauen kann. Dann musste ich wiedrum die Antwort mit dem bestrehenden COde ersetzen und kontrollieren, ob es fumktioniert. We4nn nicht, musste ich alles wieder rückgängig machen - dabei verlor ich eine MEnge Zeit.
